[프로그래머스 구명보트 문제](https://school.programmers.co.kr/learn/courses/30/lessons/42885)

```
def solution(people, limit):
    people.sort()
    count = 0
    i = 0
    j = len(people) - 1
    while i <= j:
        count += 1
        if people[j] + people[i] <= limit:
            i += 1
        j -= 1
    return count
```
