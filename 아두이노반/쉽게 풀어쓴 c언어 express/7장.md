while(조건식)
	문장;

```
int main()
{
	int n, i = 1, a = 0;
	scanf("%d", &n);

	while (i <= n) {
		a = a + i;
		i++;
	}
	printf("%d", a);
	return 0;
}
```

두수의 최대 공약수 구하기

유클리드 알고리즘

1.두 수 가운데 큰 수를 x, 작은 수를 y라 한다.

2.y가 0이면 공약수는 x와 같다.

3.r << x % y

4.x << y

5. y << r
6. 
7.단계 2.로 되돌아간다.

공약수 구하는 공식

```
int main()
{
	int x, y, r,s;
	scanf("%d %d", &x, &y);
	if (x < y) {
		s = x;
		x = y;
		y = s;
	}
	while (y != 0) {
		r = x % y;
		x = y;
		y = r;
	}
	printf("공약수는 %d 입니다.", x);

}
```
