피보나치 수 구하기

```
#구하고 싶은 n번째 숫자
n = int(input())
count = 0
n1 , n2 = 0, 1

while count < n:
    nn = n1 + n2
    n1 = n2
    n2 = nn

    count = count + 1

print("{}번째 숫자는 {} 입니다." .format(n , n1))
```

다이나믹 알고리즘으로 풀기

```
a = [1, 1]
n = int(input())
for i in range(n-2):
    a.append(0)
print(a)
for i in range(2, n):
    a[i] = a[i-1] + a[i-2]
print(a)
print(a[n-1])
```

주석 모드

```
# 피보나치 수의 첫번째, 두번째 수는 1
a = [1, 1]
# 몇번째 수를 구할 것 인지 받기
n = int(input())
# 이 다음 코드에서 범위를 벗어나지 않도록 미리 수를 추가하는 과정
for i in range(n-2):
    a.append(0)
# 잘 추가 되었는지 확인
print(a)
# 피보나치 수 계산
for i in range(2, n):
    a[i] = a[i-1] + a[i-2]
# 값 출력
print(a[n-1])
```
