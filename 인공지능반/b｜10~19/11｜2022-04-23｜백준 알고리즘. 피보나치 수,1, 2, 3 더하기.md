피보나치 수 구하기

```
#구하고 싶은 n번째 숫자
n = int(input())
count = 0
n1 , n2 = 0, 1

while count < n:
    nn = n1 + n2
    n1 = n2
    n2 = nn

    count = count + 1

print("{}번째 숫자는 {} 입니다." .format(n , n1))
```

다이나믹 알고리즘으로 풀기

```
a = [1, 1]
n = int(input())
for i in range(n-2):
    a.append(0)
print(a)
for i in range(2, n):
    a[i] = a[i-1] + a[i-2]
print(a)
print(a[n-1])
```
