미로찾기 제작중

```
from collections import deque
x = 0
y = 0

# 이중 리스트
arr = []
for i in range(3):
    arr.append(list(map(int, input().split())))
print(arr)
# 123 654 789
deq = deque()
deq.append([0,0])

arrows = [[0,1], [0,-1], [1,0], [-1,0]]

while True:

    if len(deq) == 0:
        break
    x, y = deq.popleft()

    for arrow in arrows:
        a_x, a_y = arrow

        b_x = a_x + x
        b_y = a_y + y

        if b_x < 3 and b_x >= 0 and b_y < 3 and b_y >= 0:
            if arr[b_x][b_y] == 0:
                deq.append([b_x, b_y])
                break
```  

deq 란?

Deque 의 약자

Deque 란?

Deque(데크)는 double-ended queue 의 줄임말로, 앞과 뒤에서 즉, 양방향에서 데이터를 처리할 수 있는 queue형 자료구조를 의미한다.
